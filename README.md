<h2 align="center">
  Para rodar o projeto
</h2>

```bash
  #  clone o repositório
  git clone https://github.com/fera765/Gostack-typeorm-upload.git

  #  navegue até o repositório clonado
  cd Gostack-typeorm-upload

  #  baixe as dependências
  yarn

  #  divirta-se!
  yarn dev:server ou npm dev:server
```
